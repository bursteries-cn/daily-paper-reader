## 摘要
语言模型通常分阶段进行适配：先进行公共技能阶段，接着是私有记忆阶段，最后是安全阶段，学习拒绝与所记忆实体相关的输出。在安全阶段之后撤销记忆与直接减去记忆更新的问题不同：后续的安全优化器已经改变了记忆方向。我们引入了过程边车，一个双系数编辑族 $\hatθ(λ,γ)=θ_{\mathrm{AMS}}-λΔ_{\mathrm{M}}-γ\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}$，其中 $\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}=\hat{J}_{\mathrm{S},\varepsilon}(Δ_{\mathrm{M}})-Δ_{\mathrm{M}}$，$\hat{J}_{\mathrm{S},\varepsilon}$ 是通过实际未来AdamW安全训练过程的中心割线。实现中取 $\varepsilon=1$，在自然记忆编辑尺度上；它重用 $θ_{\mathrm{AMS}}$ 作为正端点，并在 $θ_{\mathrm{A}}-Δ_{\mathrm{M}}$ 处计算一条额外的安全轨迹。我们证明了两点。第一，精确边车（使用真实传输方向 $R_{\mathrm{S}\leftarrow\mathrm{M}}$ 而非割线估计），在 $(λ,γ)=(1,1)$ 时，能够二阶近似地恢复反事实的安全专属预言机 $θ_{\mathrm{AS}}$；该证明将AdamW视为参数、一阶矩和二阶矩上的增广状态映射。第二，这种过程信息是必要的：当未来安全训练弯曲记忆方向时，所有标量任务算术编辑都会留下了一阶反事实误差，而过程边车编辑则是二阶精确的。在三个模型上，验证集选择的二维编辑在所有试验中均优于朴素任务算术的留出拒绝闭合性，并且在所有配对试验中优于 $γ=λ$ 过程-JVP子族（即缓存二维网格的对角切片）。

## Abstract
Language models are often adapted in stages: a public skill phase, a private memory phase, and a later safety phase that learns to refuse outputs tied to the remembered entities. Revoking the memory after the safety phase is not the same problem as subtracting the memory update: the later safety optimizer has transported the memory direction. We introduce process sidecars, a two-coefficient edit family $\hatθ(λ,γ)=θ_{\mathrm{AMS}}-λΔ_{\mathrm{M}}-γ\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}$, with $\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}=\hat{J}_{\mathrm{S},\varepsilon}(Δ_{\mathrm{M}})-Δ_{\mathrm{M}}$, where $\hat{J}_{\mathrm{S},\varepsilon}$ is a centered secant through the realized future AdamW safety-training process. The implementation uses $\varepsilon=1$ at the natural memory-edit scale; it reuses $θ_{\mathrm{AMS}}$ as the positive endpoint and computes one additional safety trace at $θ_{\mathrm{A}}-Δ_{\mathrm{M}}$. We prove two things. First, the exact sidecar, using the true transported direction $R_{\mathrm{S}\leftarrow\mathrm{M}}$ rather than the secant estimate, at $(λ,γ)=(1,1)$ recovers the counterfactual safety-only oracle $θ_{\mathrm{AS}}$ up to second order; the proof treats AdamW as an augmented-state map over parameters, first moments, and second moments. Second, this process information is necessary: whenever future safety training bends the memory direction, every scalar task-arithmetic edit leaves first-order counterfactual error, while the process-sidecar edit is second-order accurate. Across three models, the validation-selected 2D edit improves held-out refusal closure over naive task arithmetic in all trials, and over the $γ=λ$ process-JVP subfamily, the diagonal slice of the cached 2D grid, in all paired trials.

### 论文总结：Revocable Learned State via Process Sidecars (通过过程边车实现可撤销的学习记忆)

#### 1. 核心问题与整体含义（研究动机和背景）
- **问题**：语言模型在部署前常经历多阶段训练：先学习公共技能（阶段A），再学习私有记忆/事实（阶段M），最后进行安全训练（阶段S），学习拒绝输出与记忆相关的内容。当需要在安全训练之后撤销阶段M中学习的记忆时，简单地减去记忆更新（即朴素任务算术）会失败，因为安全优化器已经“传输”或“弯曲”了记忆方向，导致记忆方向在最终模型中不再是原始的增量，而是被安全训练过程改变。
- **背景**：传统的权重空间编辑方法（如任务算术）假设记忆方向是静态的，忽略了后续训练过程的动态影响。机器遗忘（unlearning）方法通常针对单一训练历史，未考虑安全训练对记忆的扭曲。本文提出一种新的编辑家族——**过程边车（process sidecars）**，通过估计安全训练过程对记忆方向的传输效应，实现更准确的记忆撤销，同时保留安全训练建立的拒绝行为。

#### 2. 方法论：核心思想、关键技术细节
- **核心思想**：将安全训练视为一个确定性映射 T，最终模型 θ_AMS = T(θ_AM) = T(θ_A + Δ_M)，其中 Δ_M 是记忆更新。朴素任务算术错误地假定 T 的雅可比矩阵为单位阵，但实际传输方向为 DT(θ_A)Δ_M。过程边车通过估计这个传输方向，构造一个二维编辑族，纠正传输误差。
- **关键技术细节**：
  - **传输方向估计**：使用中心割线估计安全训练过程的雅可比-向量积（JVP）：  
    Ĵ_{S,ε}(Δ_M) = (T(θ_A + εΔ_M) - T(θ_A - εΔ_M)) / (2ε)  
    实现中取 ε=1，正端点即为 θ_AMS（已知），只需额外计算负端点 T(θ_A - Δ_M)。
  - **编辑族**：定义残差边车 R̂_{S←M} = Ĵ_{S,ε}(Δ_M) - Δ_M，然后搜索二维参数 (λ, γ)：  
    θ̂(λ, γ) = θ_AMS - λΔ_M - γ R̂_{S←M}  
    该族包含朴素任务算术（γ=0）和过程-JVP线（γ=λ）。通过固定、无预言的验证规则（约束遗忘、技能保留，最大化拒绝边际）选择最优 (λ, γ)。
  - **理论保证**：
    - 精确边车（λ=γ=1，使用真实传输方向）可二阶近似地恢复反事实安全预言机 θ_AS = T(θ_A)。
    - 当安全训练弯曲记忆方向时，任何标量任务算术编辑（仅沿 Δ_M 方向）都会留下一阶误差，而过程边车达到二阶精度。
    - 二维族相对于过程-JVP线的优势来自二阶曲率效应，由验证选择实现。

#### 6. 主要结论与发现
- **过程边车有效**：验证选择的二维编辑族在所有60次试验中，在留出拒绝闭合性上优于朴素任务算术（平均提升0.33-0.44），同时保持秘密可区分性相近（校准AUC差距微小）。
- **优于过程-JVP线**：二维族同样在所有60次试验中胜过滤γ=λ的二维过程-JVP子族（包括等候选数控制实验），表明离轴（off-diagonal）修正提供了额外收益。
- **梯度上升遗忘微调（FT-unlearn）失败**：FT-unlearn导致拒绝边际大幅下降（远低于安全预言机），符合理论预测（安全训练使一阶安全梯度变小后，记忆上升方向的正安全曲率会导致二阶安全损失）。
- **理论必要性验证**：稀疏化记忆增量（TIES/DARE）仍无法恢复传输方向，确认了一阶误差必须通过过程信息修正。
- **重现性与推广性**：新鲜种子复制和8B模型上观察到一致效果，方法具有跨模型尺度和架构的泛化性。

#### 7. 优点
- **理论扎实**：提供了严格的定理证明，解释了为何朴素任务算术不足，以及过程边车如何达到二阶精度。
- **方法论简洁有效**：仅需额外一次安全训练（从负端点开始），且使用与记忆编辑相同尺度的有限差分，计算开销可控。
- **实验设计严谨**：大规模随机试验、配对统计检验、分块检验、新鲜种子复制、候选密度控制等，确保了结果的可靠性。
- **实用性**：验证选择器可在无预言机信息的情况下部署，满足遗忘、技能保留和拒绝保留的约束，适合实际应用。

#### 8. 不足与局限
- **有限尺度分析**：ε=1的割线是有限尺度估计，理论上的二阶保证是渐近的；实验证据表明在实测尺度有效，但无法保证所有设置下无意外行为。
- **特定训练场景**：方法假设安全训练是一个固定确定性轨迹（固定数据顺序、种子等）。随机性或分布偏移可能影响效果。
- **离轴系数（γ）无普适符号**：实验显示γ的选择是试验局部的（双峰分布），没有统一的缩放定律，需依赖验证选择。
- **Hessian正定性条件**：理论中关于二维族优势的严格条件需Hessian正定，但实验中仅3/8缓存探针满足，实际优势更多来自经验选择。
- **计算成本**：需要额外一次安全训练（从θ_A - Δ_M开始），虽然比完全重训代价小，但仍需一定计算资源；尤其在更大模型上可能更显著。
- **未覆盖多源撤销、重复安全训练、对抗性提取、更高秩适配器等更复杂场景**，这些被作者列为未来工作。
