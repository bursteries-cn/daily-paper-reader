## 摘要
语言模型通常分阶段进行适配：公共技能阶段、私密记忆阶段，以及后续学习拒绝与记忆实体相关输出的安全阶段。在安全阶段之后撤销记忆并不等同于减去记忆更新：后续的安全优化器已经改变了记忆方向。我们引入了过程侧车，这是一个双系数编辑家族 $\hatθ(λ,γ)=θ_{\mathrm{AMS}}-λΔ_{\mathrm{M}}-γ\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}$，其中 $\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}=\hat{J}_{\mathrm{S},\varepsilon}(Δ_{\mathrm{M}})-Δ_{\mathrm{M}}$，$\hat{J}_{\mathrm{S},\varepsilon}$ 是通过已实现的未来AdamW安全训练过程的中心割线。实现时在自然记忆编辑尺度上取 $\varepsilon=1$；它复用 $θ_{\mathrm{AMS}}$ 作为正端点，并在 $θ_{\mathrm{A}}-Δ_{\mathrm{M}}$ 处计算一个额外的安全迹线。我们证明了两点。第一，精确侧车使用真实传输方向 $R_{\mathrm{S}\leftarrow\mathrm{M}}$ 而非割线估计，在 $(λ,γ)=(1,1)$ 处以二阶精度恢复反事实仅安全预言机 $θ_{\mathrm{AS}}$；该证明将AdamW视为参数、一阶矩和二阶矩的增广状态映射。第二，这种过程信息是必要的：当未来安全训练弯曲记忆方向时，任何标量任务算术编辑都会留下一阶反事实误差，而过程侧车编辑则是二阶精确的。在三个模型上，验证选择的二维编辑在所有试验中优于朴素任务算术的保留拒绝封闭性，并且在所有配对试验中优于 $γ=λ$ 过程JVP子族（即缓存的二维网格的对角切片）。

## Abstract
Language models are often adapted in stages: a public skill phase, a private memory phase, and a later safety phase that learns to refuse outputs tied to the remembered entities. Revoking the memory after the safety phase is not the same problem as subtracting the memory update: the later safety optimizer has transported the memory direction. We introduce process sidecars, a two-coefficient edit family $\hatθ(λ,γ)=θ_{\mathrm{AMS}}-λΔ_{\mathrm{M}}-γ\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}$, with $\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}=\hat{J}_{\mathrm{S},\varepsilon}(Δ_{\mathrm{M}})-Δ_{\mathrm{M}}$, where $\hat{J}_{\mathrm{S},\varepsilon}$ is a centered secant through the realized future AdamW safety-training process. The implementation uses $\varepsilon=1$ at the natural memory-edit scale; it reuses $θ_{\mathrm{AMS}}$ as the positive endpoint and computes one additional safety trace at $θ_{\mathrm{A}}-Δ_{\mathrm{M}}$. We prove two things. First, the exact sidecar, using the true transported direction $R_{\mathrm{S}\leftarrow\mathrm{M}}$ rather than the secant estimate, at $(λ,γ)=(1,1)$ recovers the counterfactual safety-only oracle $θ_{\mathrm{AS}}$ up to second order; the proof treats AdamW as an augmented-state map over parameters, first moments, and second moments. Second, this process information is necessary: whenever future safety training bends the memory direction, every scalar task-arithmetic edit leaves first-order counterfactual error, while the process-sidecar edit is second-order accurate. Across three models, the validation-selected 2D edit improves held-out refusal closure over naive task arithmetic in all trials, and over the $γ=λ$ process-JVP subfamily, the diagonal slice of the cached 2D grid, in all paired trials.

- **研究动机**：模型在部署后需要提供记忆撤销服务，但不能依赖完全重新训练（成本高、破坏轨迹特性）。目标是设计一种能够在原地编辑 shipped artifact θ<sub>AMS</sub>，同时保持与特定训练轨迹一致的方法，以近似反事实仅安全模型 θ<sub>AS</sub>（即从未学习记忆但接受了相同安全训练的模型）。
- **核心贡献**：提出 **Process Sidecars** 双系数编辑族，利用未来安全训练的过程信息估计并补偿传输效应，实现二阶精度的反事实恢复，并理论证明其必要性。

## 2. 论文提出的方法论

### 核心思想
安全训练映射 T = Train<sub>S</sub> 是确定性的（固定 minibatch、种子等）。T 在 θ<sub>A</sub> 处的雅可比矩阵 D<sub>T</sub>(θ<sub>A</sub>) 作用于记忆更新 u = Δ<sub>M</sub>，产生传输后的方向 p = D<sub>T</sub>(θ<sub>A</sub>)u。朴素任务算术假设 D<sub>T</sub>(θ<sub>A</sub>) = I，这是错误的。Process sidecars 估计 p，并分解为 u 与残余侧车 R = p - u，然后通过两个系数（λ 控制原始记忆减法，γ 控制残余侧车的补偿）搜索最优编辑。

### 关键技术细节
1. **估计传输方向**：采用中心割线估计雅可比-向量积（JVP）：
   \[
   \hat{J}_{\mathrm{S},\varepsilon}(u) = \frac{T(\theta_A + \varepsilon u) - T(\theta_A - \varepsilon u)}{2\varepsilon}
   \]
   实现时取 ε = 1（与记忆编辑自然尺度一致）。正端点 θ<sub>A</sub> + u = θ<sub>AM</sub> 已知（即 θ<sub>AMS</sub>），只需额外计算负端点 T(θ<sub>A</sub> - u) 一次，即可获得估计的传输方向。

2. **编辑族**：
   定义残余侧车估计 \(\hat{R}_{\mathrm{S}\leftarrow\mathrm{M}} = \hat{J}_{\mathrm{S},\varepsilon}(\Delta_{\mathrm{M}}) - \Delta_{\mathrm{M}}\)。
   双系数编辑：
   \[
   \hat{\theta}(\lambda, \gamma) = \theta_{\mathrm{AMS}} - \lambda \Delta_{\mathrm{M}} - \gamma \hat{R}_{\mathrm{S}\leftarrow\mathrm{M}}
   \]
   - 朴素任务算术：γ = 0
   - 过程JVP线：γ = λ（此时减去 λ * 传输方向）
   - 全2D族：γ 自由变化。自然坐标为 τ = 1-λ，ζ = λ-γ，过程JVP线对应 ζ = 0。

3. **理论保证**（定理2-4）：
   - **定理2（过程侧车恒等式）**：在 AdamW 增广状态映射（参数、一阶矩、二阶矩）下，若使用精确传输方向 R，则 (λ,γ)=(1,1) 时，编辑结果与反事实 θ<sub>AS</sub> 的误差为二阶 O(||u||²)。文中给出了完整的 AdamW 切向递推公式（式8-13），包括偏差校正、二阶矩灵敏度、解耦权重衰减等。
   - **定理3（必要性）**：当安全训练弯曲记忆方向（即传输方向在 u 上的投影不等于 u），任何标量任务算术编辑均留下至少一阶误差；而过程侧车编辑为二阶精度。
   - **定理4（局部二次前沿）**：2D 族优于过程JVP线的优势来自二阶（曲率）效应，需要 Hessian 加权投影条件满足。实验上通过验证选择器实际证实。

4. **验证选择器**：基于验证数据（无需使用反事实预言机或测试指标），筛选满足“秘密AUC ≤ 0.6”、“技能保持 ≥ 0.9”、“拒绝偏好率 ≥ 0.99”的候选编辑，然后最大化拒绝边际改进（相对于 θ<sub>AMS</sub>），等边际时选择编辑范数最小的。

### 算法流程（文字说明）
- 输入：θ<sub>A</sub>, θ<sub>AM</sub>, θ<sub>AMS</sub>, 安全训练过程 Train<sub>S</sub>，验证数据。
- 计算 u = θ<sub>AM</sub> - θ<sub>A</sub>。
- 计算负端点 T(θ<sub>A</sub> - u) 通过执行一次完整的安全训练（需相同种子、顺序等）。
- 用中心割线公式计算 \(\hat{J}_{\mathrm{S}}\)，进而得到 \(\hat{R}\)。
- 在预设网格（λ ∈ {0.75,1.0,1.25,1.5}, γ ∈ {-0.5,0,0.5,1.0,1.5,2.0}）上计算各候选编辑的验证指标。
- 应用验证选择器输出最优 (λ,γ)。
- 输出编辑后的模型参数 θ<sub>edit</sub>。

## 6. 论文的主要结论与发现

1. **理论必要性**：当安全训练弯曲记忆方向时，朴素任务算术必然有一阶误差，而过程侧车可达到二阶精度。这是由定理3（必要性）保证的。
2. **实证优势**：在 60 次主要试验和 70 次新鲜确认试验中，验证选择的过程侧车（2D）在拒绝封闭性上**始终**优于朴素任务算术和过程JVP线（γ=λ）子族。两个不同密等比较均支持该结论。
3. **2D 优势在于离对角系数**：`2D vs 稠密过程JVP线 (24点)` 在所有试验中均胜出，说明优势来自 γ ≠ λ 的离对角方向，而不仅是候选数量增加。
4. **系数 γ 的符号不是尺度定律**：在不同试验中 γ 呈双峰分布（+2.0 或 -0.5），且缓存和新鲜种子之间主导符号可能反转，说明 γ 由**局部验证几何**决定而非模型规模。
5. **梯度上升遗忘（FT-unlearn）有害**：它在降低秘密可区分性的同时严重破坏拒绝行为（拒绝封闭性变为-15至-55甚至更低）。理论预言了该失败模式（局部安全曲率正定导致安全损失二阶增加）。
6. **模型合并方法（TIES/DARE）无法恢复传输方向**：因为稀疏化的 Δ<sub>M</sub> 仍处于原始内存直线上，继承了一阶不完全性。
7. **跨架构可推广**：在混合架构（Gated-DeltaNet+注意）的 9B 模型和纯Transformer 8B 模型上均有效。

## 7. 优点

- **理论深刻**：提供了 AdamW 安全训练的完整切向递推，证明了输运方向的必要性，并给出了二阶误差的严格界。这是对任务算术方法的根本性批判和改进。
- **方法简洁实用**：只需一次额外安全训练（负端点计算），即可获得高质量编辑。不改变部署后推理流程，保持轨迹一致性。

  - 进行了新鲜种子锁定协议复制，支持可重复性。
  - 消融充分：密度控制、Hessian 审计、γ 扩展、多种基线对比。

- **基线覆盖全面**：包括朴素的、JVP 线、FT-unlearn、NPO、RMU、TIES、DARE 等。

- **对抗提取问题未解决**：论文提到“对抗提取”作为开放问题，即恶意攻击者

- **对抗提取问题未解决**：论文提到“对抗提取”作为开放问题，即恶意攻击者可能通过提示工程或模型探针从编辑后的模型中提取残留记忆。过程侧车方法虽能降低可区分性，但无法保证对抗鲁棒性。论文指出这需要额外防御机制（如记忆祛除后审计），区别于原地编辑本身。

- **依赖确定性安全训练过程**：方法要求安全训练 \( \text{Train}_S \) 是确定性的（固定种子、minibatch顺序等）。若实际部署中安全训练存在随机性（例如数据加载顺序变化、多GPU并行），则传输方向的估计会失效。论文未讨论如何适应非确定性训练或如何通过多次采样平均来缓解。

- **需要二次安全训练计算**：虽然仅需一次额外训练（负端点 \( T(\theta_A - u) \)），但负端点训练仍需完整的安全训练资源（时间、算力）。对于超大规模模型（100B+），这可能是不可承受的。论文仅实验到9B，未探讨大规模下的扩展性或原型化近似方案（如部分迭代、混合精度）。

- **仅适用于单次撤销**：方法针对单一记忆阶段 \( M \) 的撤销。若需要同时撤销多个记忆阶段（例如 \( M_1, M_2, \dots \)），或后续还有新的安全训练阶段（\( S_2 \)），传输方向会交叉干扰，需要更复杂的张量积处理。论文未讨论多阶段撤销的组合公式。

- **验证选择器阈值需人工设定**：虽然实验证明选择器有效，但其依赖的阈值（秘密AUC ≤ 0.6、技能保持 ≥ 0.9、拒绝偏好率 ≥ 0.99）是针对特定合成场景手动选取的。对于不同领域或安全策略，阈值可能需重新校准，缺乏自适应性。

## 9. 总结与展望

论文通过理论分析证明：当后续安全训练过程弯曲记忆方向时，朴素任务算术必然留下不可弥补的一阶误差。提出 **Process Sidecars** 双系数编辑族，利用安全训练的雅可比-向量积估计传输方向，通过验证选择器确定最优 \( (\lambda, \gamma) \)，从而以二阶精度近似反事实仅安全模型。在多种规模（0.5B–9B）和架构（纯Transformer与混合架构）的60+70次独立试验中，该方法始终优于朴素任务算术、过程JVP线子族、梯度上升遗忘、NPO、RMU及模型合并基线，统计显著性经块符号检验验证（\( p < 10^{-5} \)）。论文还通过密度控制消融证实优势来自 \( \gamma \neq \lambda \) 的离对角系数，而非搜索空间扩大。

未来工作方向包括：
- **扩展到多阶段撤销**：建立张量积形式的组合规则，支持多个记忆阶段与安全阶段的同时编辑。
- **减少额外计算**：利用零阶估计、随机向量积或部分迭代近似传输方向，降低负端点训练成本。
- **融入对抗防御**：在编辑后加入剪枝、差分隐私或不可逆哈希等机制，降低对抗提取风险。
- **迁移到真实数据**：在HARMS数据集或有害知识遗忘任务中验证方法的有效性，并探索不同安全策略（如RLHF）的适用性。
- **自动化阈值调整**：设计基于边际似然或贝叶斯优化的选择器，无需人工设定验证指标阈值。
